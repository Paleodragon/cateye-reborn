Design

This business card was designed for a freelance musician and horn instructor whose primary instrument is a silver french horn. The emphasis was on giving her a clean design that calls attention to her orchestral background and professionality.

In this inspirational poster, the monochrome background with striking orange font is meant to emphasize the passionate subject, and long shadow is used to create depth, drawing the eyes to the statement and adding an element of mystery.

Development

The Rogue Pickings webpage was a class project focusing on the conversion of a PSD into a fully functional, one-page website. Attention was paid to clean code structure and pixel perfect design to produce a seamless product.

The Accererate website was a Skillcrush project whose goal was to create a 4-page WordPress website complete with Home, Blog, Contact, and Work pages. It was done via hand-coded HTML/CSS, PHP, JavaScript, and plugin installations.
